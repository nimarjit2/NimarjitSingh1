🌳 Decision Tree Regression for Financial Prediction

As part of my financial data analytics practice, I built a Decision Tree Regression model to predict Total Assets using balance sheet variables from Ashok Leyland.

Using PyCharm I trained a machine learning model to analyze how different financial factors influence the company’s asset structure.

🔎 Project highlights:

Data preprocessing and feature selection from balance sheet data

Train-test split to evaluate model performance

Implementation of Decision Tree Regression to predict Total Assets

Visualization of the decision tree structure to interpret model logic

📊 Key aspects of the model:

Decision nodes identify the most influential financial variables

Model performance evaluated using R² score and RMSE

Tree visualization helps explain how financial indicators drive predictions

💡 Key takeaway:
Machine learning models like decision trees can help uncover non-linear relationships in financial data, making them useful tools for financial analysis, forecasting, and data-driven decision making.

This project helped strengthen my skills in machine learning, financial analytics, and model visualization using Python.
